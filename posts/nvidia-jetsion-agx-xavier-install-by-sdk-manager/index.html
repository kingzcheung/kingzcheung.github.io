NVIDIA JETSON AGX XAVIER 在 ubuntu 20.04 刷不了机问题

目前 xavier 只支持在 18.04 上进行刷机，最新的ubuntu lts 20.04 上安装sdkmanager 后，会提示

SDK Manager No available releases for Ubuntu 20.04

事实上，sdkmanager 只支持 18.04 上开发，后面的版本都不支持。不过，我们可以假装高于18.04版本的ubuntu 为 18.04，以此来达到欺骗 sdkmanager 来走下去。

  1. 先备份 /usr/lib/os-release
sudo cp /usr/lib/os-release /usr/lib/os-release-bak
  1. 再编辑 /usr/lib/os-release,内容如下
NAME="Ubuntu"
VERSION="18.04.5 LTS (Bionic Beaver)"
ID=ubuntu
ID_LIKE=debian
PRETTY_NAME="Ubuntu 18.04.5 LTS"
VERSION_ID="18.04"
HOME_URL="https://www.ubuntu.com/"
SUPPORT_URL="https://help.ubuntu.com/"
BUG_REPORT_URL="https://bugs.launchpad.net/ubuntu/"
PRIVACY_POLICY_URL="https://www.ubuntu.com/legal/terms-and-policies/privacy-policy"
VERSION_CODENAME=bionic
UBUNTU_CODENAME=bionic
  1. 运行 sdkmanager

重要

当完成sdkmanager 工作后，需要恢复原来的配置：

sudo cp /usr/lib/os-release-bak /usr/lib/os-release
